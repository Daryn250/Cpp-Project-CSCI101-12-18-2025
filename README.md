# // Formatted Text to ASCII Art //
# // Daryn Grover //
Creates an ASCII art picture by getting inputs as coordinates from a text file. 
Please name your text file “readfromme.txt” and place it in the directory with the c++ program. 
Then, compile and run the program to see the results. Due to limitations, make sure that the characters are all on the same line.

# Files
plaintextcpp.txt contains the program as plain text, while main.cpp contains the program formatted with c++.

main.exe is the executable of the program.

readfromme.txt contains the data the program reads from.
